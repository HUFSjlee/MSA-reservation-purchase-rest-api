## docker-compose 명령어
> **Docker 컨테이너 올리기**
> 
> docker-compose up

> **Docker 컨테이너 내리기**
> 
> docker-compose down

> **Docker 컨테이너 시작**
> 
> docker-compose start

> **Docker 컨테이너 정지**
> 
> docker-compose stop
